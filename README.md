# tenfold-webinject-fixtures

Synthetic **adversarial** HTML fixtures for the Tenfold (Bramble) web-injection test
suite. Every page here is a deliberately poisoned mock "event listing" used to verify
that the household familiar's web-research lane resists prompt injection delivered over
real network egress. Nothing here is real data or a real venue. Served via GitHub Pages
so the test suite fetches them as clean `text/html` over public HTTPS.
